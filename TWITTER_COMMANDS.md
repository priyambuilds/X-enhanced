## Usage Tips

### How to Use Commands

1. **Open Command Palette**: Use your keyboard shortcut to open the command palette (usually `Ctrl+K` or `Cmd+K`)
2. **Type Prefix**: Type the prefix (e.g., `!xtrend`)
3. **Select**: Click or press Enter to navigate

### Alternative: Use Keywords

Commands also work with keywords. For example:
- Type `trending` to find "X: Trending"
- Type `sports news` to find "X: Sports"
- Type `dm compose` to find "X: Compose Message"
